A Python-based automated security auditor for Ethereum Smart Contracts using Slither.

## Features
- Scans Solidity files for common vulnerabilities (Reentrancy, Integer Overflow, etc.)
- Generates detailed JSON/Text reports.
- Includes sample vulnerable contracts for testing.

## Setup
1. Install Slither: `pip3 install slither-analyzer`
2. Run the scanner: `python3 scripts/scanner.py`
